### O QUE FUNCIONA

---

- **Criar / Editar / Excluir**
    - O usuário é capaz de cadastrar Clientes, Condutores, Veículos e Deslocamentos.
    - Após finalizar o cadastro, o usuário recebe a informação de cadastro com sucesso.
    - O usuário é capaz de editar Clientes, Condutores, Veículos e Deslocamentos.
    - Após finalizar a edição, o usuário recebe a informação de que a edição foi realizada com sucesso.
    - O usuário é capaz de excluir Clientes, Condutores, Veículos e Deslocamentos.
    - Antes de excluir, o usuário recebe a pergunta se realmente deseja continuar com a operação.
    - Após finalizar a exclusão, o usuário recebe a informação de que a exclusão foi realizada com sucesso.

- **Responsividade**
    - O projeto foi desenvolvido para celulares e desktop.
